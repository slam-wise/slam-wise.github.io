# Senior Applied Scientist & Machine Learning Engineer

Welcome to my portfolio. This site acts as a central hub for my technical write-ups, open-source framework documentation, and research into machine learning systems.

## 🔬 Professional Focus

I am a Senior Applied Scientist and ML Engineer with a Ph.D. in Computational Materials Science and an M.S. in Computer Science. My work centers on building end-to-end ML systems, generative AI applications, and production data pipelines designed for high-variability datasets.

My expertise encompasses:

* **Applied Science & ML Theory:** Developing robust statistical models, computer vision architectures, and graph neural networks for physical and industrial systems.
* **Generative AI & Evaluation:** Architecting agentic orchestration workflows, RAG diagnostics tools, and automated offline evaluation frameworks.
* **Production MLOps:** Building high-throughput ETL data pipelines, instrumentation frameworks, and CI/CD workflows optimized for scalability and reproducibility.

## 🛠️ Featured Projects

* **[LLM-RAG-Eval](https://github.com/slam-wise/llm-rag-eval):** An automated offline evaluation and diagnostics framework measuring key RAG performance capability metrics.
* **[Crystal Lattice GNN](https://github.com/slam-wise/crystal-lattice-gnn):** A physics-informed Graph Neural Network pipeline modeling solid-state crystal structures under periodic boundary conditions.
* **[Industrial Anomaly Detection](https://github.com/slam-wise/anomaly-detection-mvtec):** A high-precision computer vision architecture utilizing patch embedding memory banks for automated defect inspection.
